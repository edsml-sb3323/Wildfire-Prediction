WildfireLogan.fergusonfire.generate.generate.find_min_mse_image(generated_images, satellite_data)¶

Find the generated image with the minimum Mean Squared Error (MSE) compared to satellite data.

Parameters:

  • generated_images (numpy.ndarray) – An array of generated images.

  • satellite_data (numpy.ndarray) – An array of satellite images.

Returns:

A tuple containing:

  • min_mse_generated_image (numpy.ndarray): The generated image with the minimum MSE.

  • min_mse_satellite_image (numpy.ndarray): The corresponding satellite image with the minimum MSE.

  • min_mse_gen_idx (int): The index of the generated image with the minimum MSE.

  • min_mse_sat_idx (int): The index of the satellite image with the minimum MSE.

Return type:

tuple

WildfireLogan.fergusonfire.generate.generate.generate_images(model, latent_dim, num_images, device, threshold_low=0.3, threshold_high=0.5)¶

Generate images using a trained model from random latent vectors.

Parameters:

  • model (torch.nn.Module) – The trained model with a decode method.

  • latent_dim (int) – The dimensionality of the latent space.

  • num_images (int) – The number of images to generate.

  • device (torch.device) – The device to run the model on (e.g., ‘cpu’ or ‘cuda’).

  • threshold_low (float, optional) – Lower threshold for binarizing the generated images. Defaults to 0.3.

  • threshold_high (float, optional) – Upper threshold for binarizing the generated images. Defaults to 0.5.

Returns:

Generated images after thresholding, reshaped to (num_images, 256, 256).

Return type:

numpy.ndarray

WildfireLogan.fergusonfire.generate.generate.generate_sequence(model, initial_latent_vector, num_steps, device)¶

Generate a sequence of images using a trained model starting from an initial latent vector.

Parameters:

  • model (torch.nn.Module) – The trained model with encode and decode methods.

  • num_steps (int) – The number of steps (images) to generate in the sequence.

  • device (torch.device) – The device to run the model on (e.g., ‘cpu’ or ‘cuda’).
