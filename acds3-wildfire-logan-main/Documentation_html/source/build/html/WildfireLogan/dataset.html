class WildfireLogan.fergusonfire.dataset.dataset.FireDataset(train_path, test_path)¶

Bases: object

A dataset class for handling training and testing data for fire detection.

This class loads datasets from provided file paths, detects cycles within the data, and creates pairs of data points for training and validation. It also provides data loaders for batch processing.

Parameters:

  • train_path (str) – Path to the training dataset file (numpy .npy format).

  • test_path (str) – Path to the testing dataset file (numpy .npy format).
